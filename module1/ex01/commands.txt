1) htop - отображает загруженность компьютера
2) touch <name> - создает файл с именем name
3) more <name> - выводит содержимое файла name
4) head <name> - выводит первые 10 строк файла name
5) tail <name> - выводит последние 10 строк файла name
6) gzip <name> - запаковывает файл name в архив name.gz / распаковывает при аргументе -d
7) man <command> - выводит мануал по команде command
8) sort <name1> <name2> - сортирует содержимое файлов name1 и name2 и выводит
9) pwd - выводит текущую директорию
10) date - выводит день недели, месяц, число, время, часовой пояс и год
